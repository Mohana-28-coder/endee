# AI Learning Path Generator using RAG and Endee

## 📌 Overview

This project is an AI-based study system that suggests relevant topics based on user input. It demonstrates the concept of Retrieval Augmented Generation (RAG) using vector similarity.

## 🎯 Objective

To help students identify what topics to study next by analyzing similarity between concepts and generating structured learning paths.

## 🧠 How It Works

User Input → Convert to Embedding → Store/Search using Endee → Retrieve Similar Topics → Generate Learning Path

## ⚙️ Tech Stack

* Python
* Endee (Vector Database)
* RAG (Retrieval Augmented Generation)

## 🔍 Use of Endee

This project uses Endee as a vector database to:

* Store topic embeddings
* Perform semantic similarity search
* Retrieve relevant topics efficiently

## 🚀 Features

* Learning path generation
* Semantic similarity search
* AI-based study suggestions
